
Command Key Words: 
look - gives you a description of an item
    usage: look { target }
    
move - how to navigate the world
    usage: move { target } 
    
pickup - get an item from world or container and adds it to inventory
    usage: take { item } optional[{ container }]
    
drop - put an item into the environment or in a container
    usage: drop { item } optional[{ container }]
